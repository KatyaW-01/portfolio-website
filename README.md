# Portfolio Website
Welcome to my porfolio website! Get to know me and browse my projects, blog, and experience. 
### Getting Started
1. Clone this repo to your machine
    ```bash
    git clone <your-repo-url>
    cd <your-project-directory>
    ```
2. Install necessary dependencies
    ```bash
    npm install
    ```
3. Start the server from the root directory
    ```bash
    npm run dev
    ```
4. Copy the localhost link into your browser to view the webpage<br>
http://localhost:5173/
